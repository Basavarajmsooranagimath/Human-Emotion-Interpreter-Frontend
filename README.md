# 🚀 Sentiment Scope — Real-Time Sentiment Analysis Prediction System
> **AI-powered sentiment detection** — *Understand emotions behind text instantly.*

---

## 📌 Overview

**SentimentScope** is a full-stack web application that analyzes user input text and predicts whether it expresses a **Positive 😊 or Negative 😡 sentiment** — in real time.

Unlike basic sentiment tools, SentimentScope focuses on:
- ⚡ **Speed (real-time inference)**
- 🎯 **Accuracy (ML-based prediction)**
- 📊 **Clarity (confidence score + response time)**

This project demonstrates a complete **end-to-end ML pipeline**, from data preprocessing to deployment.

---

## 💭 Why I Built This

With the explosion of user-generated content (reviews, tweets, feedback), understanding sentiment is crucial.

Most tools either:
- Are too complex  
- Or don’t show *how confident* the prediction is  

So I built SentimentScope to be:
- 🤖 **Smart (ML-powered)**
- ⚡ **Fast (real-time API)**
- 📊 **Informative (confidence + performance metrics)**

The goal is not just prediction — but **interpretable insights**.

---

## ⚙️ How It Works

### Step-by-step:

1. **Input** — User enters text  
2. **Preprocessing** — Removes noise (URLs, symbols, stopwords)  
3. **Feature Extraction** — Converts text into numerical vectors (TF-IDF)  
4. **Prediction** — Logistic Regression classifies sentiment  
5. **Response** — Returns:
   - Sentiment
   - Confidence %
   - Response time  
6. **UI Display** — Result shown instantly
